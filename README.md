# 豊の国・大分 郷土史

大分県の郷土史を、自作SVGイラストとともにたどる読み物サイトです。

古代の宇佐神宮・六郷満山から、大友宗麟と南蛮貿易、城下町の時代、そして別府温泉の近代観光史まで——「一千三百年の物語」を時代・地域・テーマの3つの軸でまとめています。

## 特徴

- **出典主義**: 記事の記述は自治体・観光協会などの公的な情報源に基づき、各記事末尾に出典リンクを明記
- **和風モダンデザイン**: 和紙×墨×朱のパレット、明朝体、自作SVGイラスト（外部画像不使用）
- **地図ナビゲーション**: 大分県のデフォルメ地図から各地域の記事へ
- **アクセシビリティ配慮**: `prefers-reduced-motion` 対応、JSなしでも全コンテンツ閲覧可能

## 技術スタック

- [Astro](https://astro.build/) — 静的サイトジェネレーター
- Content Collections（Markdown）による記事管理
- 依存フレームワークなしの軽量な素のCSS/JS

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果のプレビュー
```

OGP画像の再生成:

```bash
node scripts/generate-ogp.mjs
```
